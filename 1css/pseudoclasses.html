Pseudo classes : keywords that are used to select elements based on its state or target a specific child

  • :active :- an element bieng activated(eg. clicked) Mostly links and buttons
  • :checked :- a checkbox,option ,radio input types that are enabled
  • :default :-the default set of choice (like option in select or radio)
  • :disabled : -disaled
  • :enabled :- enables
  • :focus => the element with focus
  • :hover => an elament with the mouse hover
  • :required => a form element with the required attribute set
  • :target => the element matching the fragment
  • :visited => a link that has been visited
