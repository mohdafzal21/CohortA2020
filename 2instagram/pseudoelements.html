Pseudo Elements are used to style a specific part of an element

they start with a double colon => ::
  • :: before => creates a pseudo-element before element
  • ::after => creates a pseudo-element after the element
  • ::first-letter => to apply style to first letter of block or text
  • ::first-line => apply style to first line of block or text
